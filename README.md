## 環境構築

```
git clone git@github.com:yassun-youtube/the-hack-2022-nov-team-5.git
cd the-hack-2022-nov-team-5
npm install

npm run dev
```

## スクリプト

```
# サーバ起動
npm run dev

# lint
npm run lint

# コード整形
npm run format

# テスト
npm run test
```
